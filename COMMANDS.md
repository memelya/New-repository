## Как использовать

Пиши любые команды в поле **ENTER TASK**:

```
Analyze match: Barcelona vs Real Madrid
Predict winner: Liverpool vs Arsenal
Match analysis: Juventus vs AC Milan
```

## Как работает

Когда я (Луи) анализирую матч в Telegram:
1. Я собираю реальные данные (счёт, форма команд, статистика)
2. Записываю в `state.json` что делают агенты
3. Сайт подхватывает обновления
4. Агенты ходят, обсуждают, выдают реальный результат
