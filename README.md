# 🏢 AI Agent Office

Top-down office view with retro Dandy/Sega game aesthetic. Agents walk around and discuss tasks in comic-style dialogue bubbles.

## Features

- 🎮 Retro pixel-art top-down office (8/16-bit style)
- 🪑 Individual desks with CRT monitors for each agent
- 👾 Colored agent sprites that walk around
- 💬 Comic-style dialogue bubbles when discussing tasks
- 🖥️ CRT scanline effect & retro UI
- ✏️ Type any task → agents meet and discuss it

## How to use

1. Open `index.html` in a browser (no server required)
2. Type a task in the input field
3. Hit **GO** — watch agents walk to the meeting area and discuss
4. Each agent has a unique color and personality

## Tech

Pure HTML/CSS/JavaScript with Canvas 2D rendering. No dependencies, no frameworks, no build steps.
